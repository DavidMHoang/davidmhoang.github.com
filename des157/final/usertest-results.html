User Test Results

Please test the scrolling to see if it will follow. Does it scroll and follow? Is this a good idea?

User response: I don’t see anything “following” when
I scroll. I am not sure what I am
supposed to be looking for. I am not sure
if it’s a good idea or not because I
am not sure what is supposed to be following?
Is there supposed to be a sticky element or
a fixed element? I like the concept of a
website that is a cherry blossom tree with blossoms
being the clickable elements though.

Please test tapping on an icon. Do you think this is helpful? Do you think it can be improved?

User response: When hovering over the blossom
icons some descriptive text appear describing what
I will see when I click. I like this as a
way of communicating that the blossom is clickable
but it would be nice if the blossom also
scaled up a bit; like scale: 1.1 on hover
would be nice to better communicate the interactivity.
I also would have liked to see non-distorted images.
The stretching of the photos makes them a lot
less pleasing to look at than if they were a
bit smaller but not stretched.

Please test the sound if it is working. Is the sound needed or make a difference?

User response: The sound added a nice atmospheric touch,
but it was unclear what it’s purpose was until
I clicked on it. Then I was like, “ah okay,
it’s atmospheric.” Something to indicate the
purpose of the sound to the user would have
been nice so I could decide whether I want to
click before I click.
