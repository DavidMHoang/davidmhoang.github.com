Best Practices for Modals / Overlays / Dialog Windows

Reading this article, it helped me think of
how I can implement this into the project
that I am currently working on right now.
I agree that applications or webpages should
grab the user's attention so they will interact
with it more. Having an interactive window would be
very pleasing to the eyes and it being colorful
would have more of an impact gaining attention.
Using a focusing method and initiating the user
to interact with input is a way to initiate
a form of interaction, giving control to the user.
Reading further on Modal, I found helpful information
for the usage on mobile application, which benefits my
resource searching for solutions to building my mobile app.
I believe sizing and location on the screen is
very important so it will make it blatantly obvious
that this window upfront is meaningful or important.
Having an escape hatch and a good, descriptive
title makes it easier for the user to read.
Overall, the idea of having a modal popup
makes a page or app fun by interacting with
the user and keeping them engaged with the screen.

Form Design

When first reading the article, I thought to myself,
"the information was going to be redundant to me
and that everything I was about to read what
was already common sense." However, the more I read,
the more I realized that I made many of
these mistakes in the past when trying to create
a website. For example, when it comes to labels,
I always thought that inline labels would help assist
the user when entering texts into the input label
with examples of what should be in the input
label. The same concept applies to placeholder texts where
an example could be placed in there too. I
also thought that it would be exceptional to keep
the size fields all the same to keep it
nice and neat but it is much more preferred
to resize it accordingly. Dropdown menus I personally thought
were very neat to have too but minimizing the
use of it is the best option here. Overall,
the article was very interesting to read and very
insightful, helping me see things better with an open
mind!
